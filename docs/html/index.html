Introduction

QFlightInstruments is a Qt library containing flight instruments widgets including:

  • Airspeed Indicator (qfi_ASI)
  • Altimeter (qfi_ALT)
  • Vertical Speed Indicator (qfi_VSI)
  • Turn Coordinator (qfi_TC)
  • Attitude Indicator (qfi_AI)
  • Heading Indicator (qfi_HI)
  • Electronic Attitude Direction Indicator (qfi_EADI)
  • Electronic Horizontal Situation Indicator (qfi_EHSI)
